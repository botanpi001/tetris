# テトリス (Tetris)

リッチなUIと標準的なゲームプレイ（SRS、Tスピン、ロックディレイ）を備えたテトリスアプリケーションです。

## ⚠️ 重要：ゲームの始め方

このゲームは最新のJavaScript機能（ES Modules）を使用しているため、**HTMLファイルを直接ブラウザで開いても動作しません**（画面が真っ暗なままになります）。

以下のいずれかの方法で、ローカルサーバー経由で実行してください。

### 方法1: VS Codeの「Live Server」拡張機能を使う（推奨）
1. VS Codeでこのフォルダを開きます。
2. 拡張機能「Live Server」をインストールします。
3. `index.html` を右クリックし、**"Open with Live Server"** を選択します。
4. 自動的にブラウザが立ち上がり、ゲームが開始されます。

### 方法2: Node.js (`npx`) を使う
Node.jsがインストールされている場合は、以下のコマンドをターミナルで実行してください。

```bash
npx serve .
```

コマンド実行後に表示されるURL（例: `http://localhost:3000`）をブラウザで開いてください。

### 方法3: GitHub Pages (オンラインで遊ぶ)
このリポジトリはGitHub Pagesに対応しています。
設定が完了していれば、以下のURLからブラウザだけで遊ぶことができます。

`https://botanpi001.github.io/tetris/`

### 方法4: Node.jsサーバーとして実行 (server.js)
本格的なサーバーとして実行したい場合は、以下のコマンドを使用します。

```bash
npm start
```
または
```bash
node server.js
```
これにより、`http://localhost:3000` でサーバーが起動します。

## 操作方法

- **移動**: 矢印キー（← / →）
- **回転**:
    - 右回転: ↑ または X
    - 左回転: Z
- **ソフトドロップ**: ↓（速く落下）
- **ハードドロップ**: スペースキー（瞬時に固定）
- **ホールド**: Shift または C（ピースを保持/交換）
- **ポーズ**: ESCキー

## 特徴

- **SRS (Super Rotation System)**: 壁蹴りやTスピンが可能な本格的な回転システム。
- **ロックディレイ**: 接地してもすぐに固定されず、少しの間移動や回転が可能です（「Extended Placement」ルール）。
- **リッチな演出**: パーティクルエフェクト、スコアのポップアップ表示、ネオン風のビジュアル。
