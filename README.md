# Neon Tetris (ネオンテトリス)

モダンでスタイリッシュなデザインのテトリスゲームです。PC、タブレット、スマートフォンでプレイ可能です。
GitHub Pagesでホスティングされており、インストール不要ですぐに遊べます。

## 🎮 遊び方

### 操作方法

| アクション | PC (キーボード) | モバイル / タブレット (タッチ) |
| :--- | :--- | :--- |
| **左移動** | 矢印キー 左 / `←` | 左下の青いボタン |
| **右移動** | 矢印キー 右 / `→` | 右下の青いボタン |
| **右回転** | 矢印キー 上 / `↑` | 右側の緑のボタン |
| **左回転** | `Z` キー | (モバイル版はハードドロップに変更) |
| **ソフトドロップ** (ゆっくり落とす) | 矢印キー 下 / `↓` | 右側の赤いボタン |
| **ハードドロップ** (一気に落とす) | スペースキー | 左側の緑のボタン (下矢印アイコン) |
| **ホールド** (キープ) | `Shift` または `C` | 左上の「HOLD」パネルをタッチ |
| **一時停止** | `Esc` キー | 左上の「PAUSE」ボタン |

### ゲームルール
-   **ラインクリア**: 横一列をブロックで埋めるとラインが消え、スコアが入ります。
-   **ゲームオーバー**: ブロックが画面上部まで積み上がるとゲームオーバーです。
-   **T-Spin**: T字型のブロックを回転させて隙間に入れるテクニックで、高得点が狙えます。

## 📱 モバイル・タブレット対応
-   **レスポンシブデザイン**: 画面サイズに合わせて自動的にレイアウトが調整されます。
-   **タッチ操作**: 画面上の大きなボタンで快適に操作できます。
-   **未就学児向けモード**: わかりやすい色使いとアイコンで、小さなお子様でも楽しめます。

## 🛠️ 開発者向け情報

### ローカルでの実行方法
このリポジトリをクローンし、Node.js環境で実行できます。

1.  リポジトリをクローン:
    ```bash
    git clone https://github.com/botanpi001/tetris.git
    cd tetris
    ```
2.  依存関係をインストール:
    ```bash
    npm install
    ```
3.  サーバーを起動:
    ```bash
    npm start
    ```
4.  ブラウザで `http://localhost:3000` にアクセスします。
